# Maxwell-Boltzmann Distribution Visualization

Interactive visualization of the Maxwell–Boltzmann speed distribution.

## Features

- Interactive temperature control
- Particle mass control
- Real-time graph update
- Built with JavaScript and Chart.js

## Demo

This project can be hosted with GitHub Pages.

## Physics

The Maxwell–Boltzmann distribution describes the distribution of particle speeds in an ideal gas.

Formula:

f(v) = 4π ( m / (2πkT) )^(3/2) v² exp(-mv² / 2kT)
